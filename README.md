========================================
🚀 TG PROXY - Инструкция
========================================

1️⃣ Запусти TGProxy.exe
2️⃣ Открой Telegram Desktop
3️⃣ Перейди в Настройки → Продвинутые → Тип подключения
4️⃣ Нажми "Добавить прокси" и введи:

   📡 Тип: SOCKS5
   🌐 Адрес: 127.0.0.1
   🔌 Порт: 1080
   🔑 Имя/Пароль: (оставь пустыми)

5️⃣ Нажми "Сохранить" и пользуйся!

========================================
Статистика доступна по адресу если убираете софт в другое окно:
http://localhost:8080
